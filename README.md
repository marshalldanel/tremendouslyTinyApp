# Tremendously Tiny URL Shortener

This app takes in a great big full length URL and makes it both tiny and great again.

## Getting Started



### Prerequisites

Uses Expressjs...

```
Expresssssss
```

### Installing

Start with node

```
npm init
```

Install express

```
 npm install express --save
```

Boom. Fantastic.
